# Memo Capsule Chrome 扩展

这是 `Memo Capsule` 的 `Chrome` 扩展版本。

当前版本的实际行为是：

- 默认进入 `ChatGPT / Claude / Gemini` 页面时，会在右侧显示小猫入口
- 右下角保存角标会把当前会话先保存到本地归档
- 展开面板后可以写一条临时便签，或切到一条随机书摘
- 不想显示任何内容时，可以点 `清空` 回到小猫态
- 书摘卡片右下角还有两个快捷按钮：`↻` 随机刷新一条，`→` 临时切回小猫态
- `历史存档` 里可以看完整对话、改标题、导出 `MD / TXT`
- `便签记录` 里可以编辑、删除、批量导出便签

## 最简单安装

1. 打开 `chrome://extensions`
2. 打开右上角 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择这个 `extension/` 文件夹

## 使用

1. 打开一段 `ChatGPT / Claude / Gemini` 对话
2. 先往上滚几次，确保更早的消息已经加载出来
3. 点击缩起卡片右下角保存角标，或直接点浏览器工具栏扩展图标
4. 如果想看完整对话，展开面板后点 `历史存档`
5. 如果想记一条自己的话，展开面板后在输入框里写，点 `保存`
6. 如果不想显示便签，可以点 `换一条随机书摘`
7. 如果想回到纯小猫状态，点 `清空`

## 当前规则

- 角色标签统一显示为 `YOU` 和 `AI`
- 书摘只负责展示，不进入便签历史，也不参与便签导出
- 书摘当前统一显示为 `书摘精选 · 书摘精选`，正文内容来自 corrected 精选源文件
- 归档标题支持手动改名，改名后导出文件名和 Markdown 标题也会跟着变
- 对话归档上限 `100` 条，便签记录上限 `100` 条
- 达到 `80` 条会提示你导出清理，达到 `100` 条会自动替换最早的一条
- 所有数据仅存储在你的本地设备

## 分享给别人

直接把整个 `extension/` 文件夹打包发给别人即可。

对方解压后，同样在 `chrome://extensions` 里选择 `加载已解压的扩展程序`。
