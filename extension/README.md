# AI Chat Session Exporter Chrome 扩展

这是一个本地 `Chrome` 扩展版本。

安装后它不再直接下载文件，而是先把当前会话保存到本地归档。

现在的用法是：

- 点击浏览器工具栏里的扩展图标：一键保存当前会话到本地归档
- 或点击页面里的缩起便签入口：打开小面板，点 `保存当前`
- 打开 `归档` 后，可以在页面里直接浏览完整上下文，再决定导出 `MD` 或 `TXT`

## 最简单安装

1. 在 `Chrome` 地址栏输入 `chrome://extensions`
2. 打开右上角 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择这个文件夹：

`extension/`

5. 安装完成

## 使用

1. 打开 `ChatGPT / Claude / Gemini` 的某一段对话
2. 先向上滚动几次，确保旧消息已经加载出来
3. 任选一种方式保存当前会话：
   - 点击 Chrome 工具栏扩展图标
   - 点击页面上的缩起便签入口，再点 `保存当前`
4. 想查看时，点开缩起便签入口 → `打开归档`
5. 在归档里：
   - 可直接浏览保存下来的完整上下文
   - `导出 MD`：保留标题、层级、代码块，更适合再喂给 AI
   - `导出 TXT`：纯文本更干净，适合粘到不支持 Markdown 的地方

## 交互说明

- 默认是收起的便签入口，不再是两个悬浮大按钮
- 缩起状态和展开状态都可以拖动位置
- 你可以把缩起入口改成任意一句提醒或鼓励语
- 面板下方保留了 `@一龙小包子` 的外链入口

## 分享给别人

你可以直接把整个文件夹打包发给别人：

`extension/`

别人解压后，同样去 `chrome://extensions` 里选择“加载已解压的扩展程序”即可。
