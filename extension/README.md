# Memo Capsule Chrome 扩展

这是一个本地 `Chrome` 扩展版本。

安装后它不再直接下载文件，而是先把当前会话保存到本地归档。

现在的用法是：

- 点击浏览器工具栏里的扩展图标：一键保存当前会话到本地归档
- 默认进入大模型网页时会显示小猫入口；点小猫展开
- 如果你已经设了 memo 或随机书摘，缩起态会显示内容卡片，右下角小角标可直接保存当前会话
- 想编辑 memo、切换随机书摘或查看历史时，点缩起入口本体展开
- 打开 `归档` 后，可以在页面里直接浏览完整上下文，再决定导出 `MD` 或 `TXT`

## 最简单安装

1. 在 `Chrome` 地址栏输入 `chrome://extensions`
2. 打开右上角 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择仓库里的 `extension/` 文件夹

5. 安装完成

## 使用

1. 打开 `ChatGPT / Claude / Gemini` 的某一段对话
2. 先向上滚动几次，确保旧消息已经加载出来
3. 任选一种方式保存当前会话：
   - 点击 Chrome 工具栏扩展图标
   - 在有 memo / 书摘的缩起卡片右下角点保存角标
4. 想查看时，点开缩起入口本体 → `打开归档`
5. 在归档里：
   - 可直接浏览保存下来的完整上下文
   - `导出 MD`：保留标题、层级、代码块，更适合再喂给 AI
   - `导出 TXT`：纯文本更干净，适合粘到不支持 Markdown 的地方
6. 如果要管理你自己写的 memo：
   - 在面板里点 `便签历史`
   - 支持按时间查看、编辑、删除、选择后批量导出

## 交互说明

- 没有 memo 时只显示小猫入口；有 memo 或随机书摘时只显示内容卡片，不再重复显示小猫
- 缩起状态和展开状态都可以拖动位置；拖动不会再误触发展开
- 便签历史只记录你手写的 memo；随机书摘只负责展示，不进入历史和导出
- 面板下方保留了 `@一龙小包子` 的外链入口

## 分享给别人

你可以直接把仓库里的 `extension/` 文件夹打包发给别人。

别人解压后，同样去 `chrome://extensions` 里选择“加载已解压的扩展程序”即可。
