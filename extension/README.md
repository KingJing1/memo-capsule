# Memo Capsule Chrome 扩展

`Memo Capsule` 的 Chrome 扩展版本。

## 当前行为

- 进入 `ChatGPT / Claude / Gemini` 页面时，右侧默认显示小猫入口
- 点击保存按钮，会把当前会话保存到本地归档
- 展开面板后可以写便签，或切到一条随机书摘
- `历史存档` 里可以查看完整对话、改标题、导出 `MD / TXT`
- `便签记录` 里可以编辑、删除、批量导出便签
- 书摘折叠态右下角支持 `↻` 随机刷新，`→` 临时切回小猫

## 安装

1. 打开 `chrome://extensions`
2. 打开右上角 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择这个 `extension/` 文件夹

## 使用

1. 打开一段 `ChatGPT / Claude / Gemini` 对话
2. 往上滚几次，确保更早的消息已经加载出来
3. 点击右侧保存按钮，把当前对话存入本地归档
4. 点小猫打开面板，使用便签、书摘、历史存档或便签记录

## 规则

- 角色标签统一显示为 `YOU` 和 `AI`
- 对话归档上限 `100` 条，便签记录上限 `100` 条
- 达到 `80` 条会提示导出清理，达到 `100` 条会自动替换最早的一条
- 随机书摘只负责展示，不进入便签历史，也不参与便签导出
- 所有数据仅存储在你的本地设备

## 分享给别人

直接把整个 `extension/` 文件夹打包发给别人即可。
