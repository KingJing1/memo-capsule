# Memo Capsule

存下对话，记住想法，留住自己。

---

## 这是什么

一个浏览器插件。装上之后，ChatGPT / Claude / Gemini 的对话界面右侧会多一只小猫咪。

点小猫咪，打开面板。点旁边的保存按钮，把当前对话先存到本地归档，再在面板里查看、改名、导出 Markdown 或 TXT。

面板里还有两样东西：一个随手便签，一个随机书摘。等 AI 思考的时候，记一笔，或者翻到一句话。

三个功能都很轻。它们指向同一件事：在被 AI 生成内容包围的界面里，保留一点属于你自己的东西。

纯本地存储，不上传数据，没有账号。

## 功能

**对话存档** — 一键保存当前对话到本地归档，在历史里查看、改标题、导出 Markdown / TXT。支持 ChatGPT、Claude、Gemini

**随手便签** — 记下等会要问的、要做的，折叠后常驻显示，也可以进入便签记录编辑、删除、批量导出

**随机书摘** — 在没有便签时切换成一条随机书摘，折叠态可直接换一条，也可以一键切回小猫

## 安装

1. 下载本仓库（Code → Download ZIP 或 `git clone`）
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角打开「开发者模式」
4. 点击「加载已解压的扩展程序」，选择 `extension/` 文件夹
5. 打开 ChatGPT / Claude / Gemini，右侧出现一只小猫咪

## 使用

**保存对话** — 点击小猫咪右下角的保存按钮，当前对话会保存到本地归档。在面板里点「历史存档」查看、改名和导出。

**写便签** — 点小猫咪打开面板，写一句话，点「保存」。折叠后便签替代小猫咪常驻显示。

**看书摘** — 展开面板后点「换一条随机书摘」，折叠态显示一条随机书摘。书摘卡片右下角可以继续随机刷新，或临时切回小猫。

## 架构

**`extension/content.js`** — 主逻辑。面板 UI、对话解析、存储管理、导出、便签和书摘都在这里。

**`extension/excerpts.js`** — 随机书摘数据。

**`extension/assets/cat-save.png`** — 页面里的小猫入口。

**`extension/manifest.json`** — Chrome 扩展配置。

为什么主逻辑集中在一个文件里：这个插件的复杂度还不值得拆。样式内联，HTML 用 JS 生成，状态存在 `chrome.storage` 里，一个文件可以看完整体行为。

## 设计哲学

1. 这是一个角落，不是一个平台。功能克制，不膨胀。
2. 等待不是空白。AI 思考的几秒钟，可以是一个小的记录窗口。
3. 入口要轻。默认只是小猫，写了便签或切到书摘才替换显示。
4. 数据只属于你。纯本地，不上传，不追踪，不注册。

## 当前规则

- 对话归档上限 `100` 条，便签记录上限 `100` 条
- 到 `80` 条会提示导出清理，到 `100` 条会自动替换最早的一条
- 角色标签统一显示为 `YOU` 和 `AI`
- 归档标题支持手动改名，改名后导出文件名和 Markdown 标题会一起更新
- 随机书摘只负责展示，不进入便签历史，也不参与便签导出

## 环境要求

Chrome 或任何 Chromium 内核浏览器
